Course editor

Use this editor to create, edit, and delete courses, tutorials, exercises, and parts.

  • When creating a course, you can add tutorials to the course.
  • When creating a tutorial, you can add exercises to the tutorial.
  • When creating an exercise, you can add variants to the exercise.
  • When creating a variant, you can add parts to the variant.
  • When creating a part, you can add a title, description, and type to the part.

Each student will receive a random variant of the exercise. The types of exercises are:

  • Numerical answer
  • Document upload

For numerical answer exercises, the student will enter a numerical answer. For document upload exercises, the student will upload a document. The student will be able to submit the answer multiple times. The student will be able to see the result of the exercise after submission.
